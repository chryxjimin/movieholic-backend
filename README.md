# MOVIEHOLIC

 Movieholic is a Single Page Application(SPA) that allows users to create reviews for movies.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'movieholic'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install movieholic
